# Platform-Specific Parameters

## TikTok

| Parameter | Type | Description |
|-----------|------|-------------|
| `tiktok_title` | String | Override title |
| `privacy_level` | String | Visibility policy |

## Instagram

| Parameter | Type | Description |
|-----------|------|-------------|
| `instagram_title` | String | Override title |
| `media_type` | String | `IMAGE`, `STORIES`, `REELS` |

## X (Twitter)

| Parameter | Type | Description |
|-----------|------|-------------|
| `x_title` | String | Override title |
| `x_long_text_as_post` | Boolean | Single post instead of thread |

## Common Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `first_comment` | String | Auto-post first comment |
| `async_upload` | Boolean | Background processing |
| `scheduled_date` | String | ISO-8601 schedule time |
| `timezone` | String | IANA timezone |
